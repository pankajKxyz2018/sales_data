# Retail Sales KPI Analysis and Visualization using Python

## Project Overview

This project performs retail sales data analysis using Python, Pandas, Matplotlib, and Seaborn. The analysis includes data cleaning, KPI calculation, customer revenue analysis, and business visualizations.

---

## Dataset Information

The dataset contains supermarket sales transaction data including:

* Order details
* Customer names
* Retail prices
* Order quantities
* Tax values
* Total sales revenue

---

## Tools & Technologies Used

* Python
* Pandas
* Matplotlib
* Seaborn
* VS Code

---

## Data Analysis Workflow

* Raw Excel data inspection
* Data cleaning and preprocessing
* Removing unnamed columns
* Missing value and duplicate checks
* KPI analysis
* Sales trend analysis
* Customer revenue analysis
* Data visualization

---

## Key KPIs Analyzed

* Total Sales Revenue
* Average Order Value
* Highest and Lowest Order Value
* Total Quantity Sold
* Total Tax Collected
* Top Revenue-Generating Customers

---

## Visualizations Created

* Daily Sales Trend Line Chart
* Top Customers Revenue Bar Chart
* Order Quantity Distribution Histogram
* Revenue Distribution Boxplot
* Sales Correlation Heatmap

---

## Business Insights

* Identified top-performing customers
* Analyzed daily sales trends
* Explored customer purchasing behavior
* Performed sales correlation analysis
